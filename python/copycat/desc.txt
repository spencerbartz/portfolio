CopyCat is a Client / Server file transfer tool. Both client and server are 
command line tools which accept various flags to manually set port etc. Be sure to use the help menu. 
Packaged with CopyCat is a logger called <strong>KennyLogger</strong>. Erros are logged on 
both the client and server side according to 5 severity levels set in the logger. <a href="https://github.com/spencerbartz/copycat">View the source code!</a>
